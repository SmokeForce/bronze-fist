Service API

This service contains a couple of endpoints which include:

  • /{{ version }}/generate/ : generates one random item
  • /{{ version }}/generate/<number>/ : generates <number> random items
  • /{{ version }}generate/<number>/<type>/ : generates <number> random items of type <type>

For example, invoking: 

https://{{ service_url }}/{{ version }}/generate/10/

will generate 10 random items. Invoking: 

https://{{ service_url }}/{{ version }}/generate/11/Containers/

will generate 11 random items of type "Containers".

The data are returned as a list of items.
